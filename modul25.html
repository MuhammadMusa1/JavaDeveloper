Слайд 2

Что такое JavaFX?

  • JavaFX: Современный фреймворк для создания красивых интерфейсов.
  • Для чего: Десктопные приложения (Windows, Mac, Linux).
  • Отличия от Swing/AWT: Простота, CSS, богатые компоненты.
  • Аналогия: Как современный офис вместо старого кабинета.

Вопрос: Почему JavaFX лучше Swing?

TableView: Таблица данных

  • Показывает данные в таблице.
  • Пример: Список сотрудников (имя, должность).
  • Аналогия: Как таблица в Excel для банка.
Слайд 5

Код: TableView

import javafx.scene.control.TableView;
import javafx.scene.control.TableColumn;
import javafx.beans.property.SimpleStringProperty;

public class Employee {
    private final SimpleStringProperty name;
    private final SimpleStringProperty role;

    public Employee(String name, String role) {
        this.name = new SimpleStringProperty(name);
        this.role = new SimpleStringProperty(role);
    }

    public String getName() { return name.get(); }
    public String getRole() { return role.get(); }
}

TableView table = new TableView<>();
TableColumn nameCol = new TableColumn<>("Имя");
nameCol.setCellValueFactory(cell -> cell.getValue().nameProperty());
TableColumn roleCol = new TableColumn<>("Должность");
roleCol.setCellValueFactory(cell -> cell.getValue().roleProperty());
table.getColumns().addAll(nameCol, roleCol);

Слайд 7

Код: ListView

import javafx.scene.control.ListView;
import javafx.collections.FXCollections;

ListView listView = new ListView<>();
listView.setItems(FXCollections.observableArrayList(
    "Кредитный отдел", "Касса", "IT"
));
listView.getSelectionModel().selectedItemProperty().addListener(
    (obs, oldVal, newVal) -> System.out.println("Выбран: " + newVal)
);

TreeView: Дерево данных

  • Показывает иерархию (ветки, листья).

Слайд 9

Код: TreeView

import javafx.scene.control.TreeView;
import javafx.scene.control.TreeItem;

TreeView treeView = new TreeView<>();
TreeItem root = new TreeItem<>("Ориёнбанк");
TreeItem dushanbe = new TreeItem<>("Душанбе");
TreeItem khujand = new TreeItem<>("Худжанд");
root.getChildren().addAll(dushanbe, khujand);
treeView.setRoot(root);
treeView.getSelectionModel().selectedItemProperty().addListener(
    (obs, oldVal, newVal) -> System.out.println("Выбран: " + newVal.getValue())
);

Дерево филиалов с выбором.

Слайд 17

Частые ошибки

  • Неправильная привязка данных в TableView.
  • Изменение GUI не из потока JavaFX.
  • Игнорирование Property (ручное обновление).

Вопрос: Почему нельзя менять GUI из другого потока?

Слайд 18

Связь с другими темами

  • Коллекции: ObservableList для TableView.
  • Многопоточность: Platform.runLater для GUI.
  • ООП: Модели (Employee) для данных.
Platform.runLater(() -> table.getItems().add(new Employee("Алишер", "Менеджер")));
